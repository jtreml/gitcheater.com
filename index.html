Showing DIFFERENCES

# Working Copy vs. Staging Area
•
  • Man Page
  • Google
  • Git Book
  • ························
  • Two vs. Three Dots
 git diff [--name-only] [file]
•
  • Man Page
  • Google
 git difftool [-t toolName] [file]
# Staging Area vs. Last Commit
git diff --cached [file]
git difftool [-t toolName] --cached [file]
# Working copy vs. commit
git diff <commit> [file]
# Commit vs. ...
git diff <commit>..<commit> [file] # other commit
git diff <commit>...<commit> [file] # common ancestor
# Differences between remote and local...
•
  • Man Page
  • Google
  • Git Book
 git fetch <remote>
git diff HEAD[..]FETCH_HEAD # ... direct diff
git diff HEAD...FETCH_HEAD # ... common ancestor

RESETting things

# Reset everything (working dir & stage to last commit)
•
  • Man Page
  • Google
  • Git Book
 git reset --hard HEAD
# Reset a single file
git checkout -- <file> # working dir to staged state
git checkout HEAD <file> # working dir & stage to last commit
